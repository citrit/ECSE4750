===============================================================
		Rendering to Texture Demo
===============================================================

Author: Jason Allen
Email: jra101@home.com
website: delphigl.cfxweb.net

===============================================================
 {Description}

 Demonstrates "rendering to texture" in OpenGL. A rotating
 torus is drawn and then copied to a texture. This texture is 
 then mapped onto the sides of a rotating cube.
 
 {Keys}

 SPACE          Toggle cube rotation
 ESC		Exit

===============================================================

Compiled and tested under Delphi 5.0 on Windows 2000 only.

Feedback for different versions of Delphi and Windows would
be greatly appreciated.


If you find this usefull please let me know or visit my site.
