 MPEG Encode Example

 

The MPEG encoder from Berkeley is available for Windows 95/NT. Here is a ZIP file of the program and an Example param file used to make an mpeg of the MriIso example. To create the sample MPEG follow these steps:

  1. Remove the comment from the line:
    	//f.savetofile = true;

    This will enable the WriteImageAsPPM method od the vtkRenderWindow

  2. Compile and run MriIso and select the menu choice for animate. This will create the PPMs.
  3. Run the mpeg_encoder using:
    	mpeg_encode Example2.param

    This will output some parameters to the screen telling the current status and create the Example2.mpg file.

Experiment with the java code to make more frames and then modify the param file to reflect these changes. Specifically look for a line that reads:

image.*.ppm [10-900+200]

This specifies the input file names and the frames which to encode. Look through the param file and let me know if there are any questions. This is similar to the mpeg_encode on the RCS system so the same parameters apply. You can run the mpeg_encoder by specifying:

setup ~citrit/mpeg_encode
mpeg_encode Example2.param

Good luck and let me know if there are any problems or questions.
