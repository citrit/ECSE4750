This course (35.6961) is offered as a graduate level topics course though the Electrical, Computer, and Systems Engineering department at Rensselaer

DESCRIPTION:

The course will cover topics associated with advanced computer graphics including Graphical Application Programmer Interfaces (APIs) such as OpenGL, hi performance graphics engines and visualization pipelines. There will be a section on cross platform development and issues involved with porting graphics software. The course will also explore algorithms for extracting visual information from data sets, such as marching cubes, used for determining iso-surfaces, contours, and cut planes. There will be a short overview of Object Oriented design and Object Oriented systems followed by a section on creating visualizations using packages such as DataExplorer and object oriented software libraries such as VTK and Open Inventor. The course will conclude with a term project in which the student models a real world phenomenon using any of the applications covered in class. Term project required. Prereqs: 35.475 Computer Graphics, 66.436 Data Structures or equivalent, some familiarity with C++.
