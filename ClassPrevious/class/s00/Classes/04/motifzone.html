Documentation for X Window System courtesy of The Motif Zone

This set of manual pages for X includes internal links, so that you can jump from one function description to another or from one data structure description to another. The original files come from the X11R6 sources and were converted to HTML for on-line reading.

The listings are alphabetical; when necessary, pages are listed twice (e.g. XtFree and XtMalloc are both listed, but both connect you to the same page, which has information on both of those functions).

Bugs:

  • updated 01-Oct-98 to include missing pages
  • there are no Motif manual pages. Motif man pages are considered source by The Open Group and may not be distributed except under licensing arrangements.
  • in an ideal environment, the header files would be HTML-ified as well, with correct pointers into the particular relevant portions of each file
  • words which originally were hyphenated don't become links. It is unclear how text should be reformatted when it is unhyhenated, and I'm not clever enough to figure it out on the fly. Ideally, the tools used to produce the pre-formatted man pages, on which these HTML pages are based, wouldn't hyphenate their output but would instead use a simple fill.
