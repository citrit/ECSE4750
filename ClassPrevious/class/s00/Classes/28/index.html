Homework 9 - Due Thurs Nov 11

  1. When would you prefer a monospaced font to a proportional font?

  2. Which is better at low resolution: a bitmapped font or an outline font? Why?

  3. Which scales better?

  4. Which is PostScript?

  5. Use the vector rotation formula to rotate the point (3,1,2) by 90 degrees around the axis (0,1,0).

  6. Rotate these 2-D Cartesian points by 90° about the point (1,1):

    (0,0), (1,1), (2,2).

  7. Find the matrix M, depending only on the vector a=(2,1,0), such that for all vectors p, Mp=a.p a.

  8. Find the matrix M, depending only on the vector a=(2,1,0), such that for all vectors p, Mp=a x p.

  9. Why can a rotation matrix never have an eigenvalue of 2?

  10. In our vector formula for rotating a point p about an axis a,

    1. Does p need to be normalized?
    2. Does a need to be normalized?

  11. I'm trying to rotate the square with opposite corners at (0,0) and (1,1) many times by a small angle, t=0.1o, by applying the usual 2D rotation matrix. cos t=0.99984770, and sin t=0.00174524, both to within 10-8. Therefore, the rotation matrix is

    0.99984770 0.00174524
    -0.00174524 0.99984770

    The problem is that after I use Matlab to apply this matrix 3600 times, which is one complete rotation, the area of the rotated square is only 0.3377, down from 1. What test on the matrix could have told me that this would happen, including giving the actual resulting area, without my having actually to rotate the points?
