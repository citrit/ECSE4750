Tcl Supported Platforms

This is from http://www.scriptics.com/products/tcltk/8.2.html

Tcl/Tk 8.2 is compatible with most releases of the following operating systems:

  • Windows 95 and 98
  • Windows NT
  • Solaris and SunOS
  • Linux
  • HP-UX
  • SGI
  • IRIX
  • Digital Unix
  • AIX
  • SCO Unix
  • NetBSD, FreeBSD, BSDi
  • Most other Unix-like operating systems
  • Macintosh (68K and Power Mac)
