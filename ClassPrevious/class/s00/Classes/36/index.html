ECSE-6800 Advanced 3-D Computer Graphics and Visualization - S00 Course

There are openings for motivated students who want to learn more about 3D computer graphics, visualization, and animation. The course Advanced Computer Graphics and Data Visualization is being offered this semester as a studio course. In this class you will learn, first hand, how to create 3D animations and simulations using the latest Object Oriented technologies C++ and Java. Also we will be using OpenGL as the graphics API, a proven indsutry standard available on all computing platforms.

The new studio format allows you to learn this technology, ask questions, and work through difficult topics in class. The use of Java will allow for cross platform capabilities and give you the skills necessary for the working world. Please respond to this email if you are interested. I need to have a count of interested students as soon as possible.

Homework 10 - Due Thurs Dec 2

  1. One advantage of homogeneous coordinates is that points at infinity can be represented. What is the homogeneous coordinate representation for the point at infinity on the line y=x+2? (Might require thinking.)

  2. For a perspective projection from the origin to the projection plane at x=5.

    1. Calculate the 4x4 homogeneous matrix.

    2. Where would these Cartesian points project to?

      1. (1,0,3)
      2. (1,1,4)
      3. (0,1,5)

    3. What point would the infinite homogeneous point (1,2,3,0) project onto?

  3. Why should you clip before projecting an object?

  4. How does the Sutherland-Hodgman patent represent vertices?

  5. What word does the patent's claims use to describe the number of individual processing units?

  6. Draw an example where the original polygon is connected, but the clipped result isn't.
