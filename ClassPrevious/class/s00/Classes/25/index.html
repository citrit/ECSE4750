Enrichment - Polarization

Today in class I'll show an oddity with polarized filters. If you place a vertically polarized filter on a horizontal one, then all light is blocked. However, if you insert a 3rd filter, rotated to 45 degrees, between them, then some light gets thru. But, if you place the 3rd filter outside the original two, then no light still gets thru.

Why should adding another filter increase the amount of light? Why does the order matter?

Homework 6 Probability Answer

The question concerned a 1000x1000 color LCD display, with 3,000,000 transistors, where 1 in 1,000,000 transistors is bad.

  1. p = probability that any given transistor is bad = 10-6.

  2. N = number of transistors = 3,000,000.

  3. probability that any given transistor is good = 1-p

  4. probability that all transistors are good = (1-p)N

  5. approximately = e-pN = e-3 = 0.050.

  6. Average number of bad transistors = L = pN = 3

  7. Probability that exactly k transistors are bad is a Poi sson distribution = P(L,k) = exp(-L) Lk/k!

  8. Use Maple to find sum(Poisson(3,k),k=10,infinity) = 0.00110
