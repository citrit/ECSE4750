3. Scientific Visualization

a) (2 Pts) What is meant by implicit and explicit control of a rendering system? Which does our rendering system use?

Explicit control is when each object informs the program controller that it has changed.

Implicit control is when the controller polls each object to be rendered to see if it has changed.

(5 Pts) Based on our discussions of marching quads, fill in the GetPoint Java algorithm to extract where a given scalar value val will fall between the given points Pt1 and Pt2 with given scalar range values Sc1 and Sc2. This algorithm should assign the calculated x,y,z to res and work for any given val .
typedef Struct Point {
	float x, y, z;
};
Point res = new Point(); 
Point& GetPoint(Point Pt1, Point Pt2,
		float Sc1, float Sc2, float val) {
	// You fill in the blank.
If (Sc1 > val && Sc2 < val)
	 {
  res = (Pt1 - Pt2)* ((val - Sc2)/(Sc1 - Sc2));
  }
else if (Sc1 < val && Sc2 > val)
	 {
  res = (Pt2 - Pt1)* ((val - Sc1)/(Sc2 - Sc1));
  } 
else
  {
  res = 0;
  }
 	return res; 
}
